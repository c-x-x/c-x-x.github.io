idea配置代理

发表于2021-07-01|更新于2021-08-18|idea
|阅读量:

打开idea

左上角Flie

File | Settings | Appearance & Behavior | System Settings | HTTP Proxy

选择Manual proxy configuration

选择socks

hostname：localhost

prot number：7890

点击Check connection

输入https://youtube.com

点击ok

如下图显示就成功了
